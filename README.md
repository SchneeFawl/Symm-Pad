# ⌨ Symm-Pad
Macropad named '**Symm Pad**', plan on using this for quick shortcuts which could be used in various applications. <br>
Hardware project (Hackpad) for [Blueprint](https://blueprint.hackclub.com/).

## ⚙ Components to be added
- 6 keys along with backlit LEDs.
- 0.91" OLED display above or beside the keys.
- A rotary encoder knob above the keys, planning to use it as volume control.
- A side LED strip light will also be present on the left and right sides of the pad.
